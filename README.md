# AI Orchestration Platform

A modular orchestration framework for building and coordinating agents using an Agent Development Kit (ADK), Agent-to-Agent (A2A) Protocol, and Model Context Protocol (MCP).

## Features
- ADK for agent creation and lifecycle management
- A2A protocols for agent communication  
- MCP for exposing and invoking remote functions, tools and prompts  
- LiteLLM adapter for LLMs and embeddings
